# LteSIM7600

Arduino library for 4G modules SIM7500/SIM7600 series.  


## SIM7500_ / SIM7600_ series:

 - SIM7500E
 - SIM7500E
 - SIM7500A
 - SIM7600C 
 - SIM7600E
 - SIM7600G

### Supported arquitecture
- ESP32
