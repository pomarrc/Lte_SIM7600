## SIM7500_ / SIM7600_ series:

 - SIM7500E
 - SIM7500E
 - SIM7500A
 - SIM7600C 
 - SIM7600E
 - SIM7600G

### Supported arquitecture
- ESP32

## Features

**Data connections**
- TCP (HTTP, MQTT)
- SSL/TLS (HTTPS, FTPS, MQTTS)

**SMS**

**Voice Calls**
- Functions:
    - Dial, hangup
    - Receiving calls
    - Incoming event (RING)

**Location**
- GPS/GNSS
